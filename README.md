# ATM

A simple console-based ATM (Automatic Teller Machine) simulation built with C#.  
The application demonstrates basic banking operations such as checking balance, depositing, and withdrawing funds.

## Features

- View account balance
- Deposit money
- Withdraw money
- Simple console UI

## Getting Started

1. Clone the repository:

git clone https://github.com/antonlidstroem/ATM.git

2. Open in Visual Studio or run via CLI:

dotnet run
